# CSS Helpers Gem
A Ruby Gem of CSS helper classes that can be used to reduce the amount of CSS your write.

## Installation

Add this line to your application's Gemfile:

    gem 'css-helpers'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install css-helpers
